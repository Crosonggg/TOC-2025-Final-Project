## How to Run

### 1) clone
```bash
git clone https://github.com/Crosonggg/TOC-2025-Final-Project.git
cd TOC-2025-Final-Project
```

### 2) Install required packages:
```bash
pip install -r requirements.txt
```
### 3) Configure API Key
將 `API.txt` 內容改成你的 API key：


### 4) run
```bash
streamlit run app.py
```
